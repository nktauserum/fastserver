# fastserver

Небольшая имплементация HTTP-сервера на Си с минимумом зависимостей. Представляет собой учебный проект, предназначенный для лучшего понимания низкоуровневых процессов под слоями абстракции. Прогресс по производительности с каждой оптимизацией отражают файлы в папке reports.

## Что реализовано

- Работа с системными сокетами и API системы в общем.
- Многопоточность (thread pool & epoll)
- Парсинг части HTTP-запроса по методу finite-state machine
- Базовый роутинг (на костылях)

## Что надо сделать

- Парсинг заголовков и тела запроса
- Кэширование статических файлов в памяти для быстрого доступа
- Нормальный и производительный роутинг, поддержка динамических маршрутов
